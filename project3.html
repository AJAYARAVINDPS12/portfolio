Full-Stack Web Development

This project is a full-stack web application built using the MERN stack (MongoDB, Express, React, Node.js). It includes a homepage, login page, signup page, and features allowing users to upload and search photos.

Features:

  • Homepage, login, and signup pages.
  • User ability to upload and search photos.
  • Responsive and dynamic UI.

Technologies Used:

  • MongoDB
  • Express
  • React
  • Node.js
