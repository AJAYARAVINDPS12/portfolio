Online Movie Rental System

This project is a fully functional online movie rental system built using Java and MySQL. Users can rent and return movies, view available movies, and process payments seamlessly.

Features:

  • Rent and return movies.
  • View available movies in different formats.
  • Process payments for rentals.
  • Manage concurrent rentals with multithreading.

Technologies Used:

  • Java
  • MySQL
  • Multithreading
